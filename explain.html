Number Systems

Take a moment to count up the number of symbols on the following line:

1 2 3 4 5 6 7 5 r s 3 4

Never mind, I'll do that for you. There are twenty seven symbols on that line. Using our standard base 10 numbering system, we write this as:

27

understanding that this means 2x10 + 7. But look at this :

27 = 10x2 + 1x7. Base 10 representation : 27
27 = 9x3 + 1x0. Base 9 : 30
27 = 8x3 + 1x3. Base 8 : 33
27 = 7x3 + 1x6. Base 7 : 36
27 = 6x4 + 1x3. Base 6 : 43
27 = 25x1 + 5x0 + 1x2. Base 5 : 102
27 = 16x1 + 4x2 + 1x3. Base 4 : 123
27 = 27x1 + 9x0 + 3x0 + 1x0. Base 3 : 1000
27 = 16x1 + 8x1 + 4x0 + 2x1 + 1x1. Base 2 : 11011

There are lots of ways to represent the number of symbols on the line.
