Number Systems

When we look at a number like 5923 we understand that each position represents a power of 10. We think in base 10. Five thousand, nine hundred and twenty three is understood to mean 3 + (2x10) + (9x102) + (5x103).  Each digit is multiplied by the base to the power of n where n is the postion of the digit in the number reading from right to left.

Now, take a moment to count up the number of symbols on the following line:


Never mind. There are twenty six symbols on that line. Using our standard base 10 numbering system, we write this as 26 because there are 2 groups of 10 symbols and 6 more symbols : 2x10 + 6. You can write this number using any base. By the way, in order to make this work, mathematics presumes that any number to the power of 0 is equal to 1:

There are at least 10 ways to represent the number of symbols shown above. Why are there no entries for base 0 or base 1? The number 1 to any power is always 1 - e.g. 13 = 1x1x1 = 1 and the number 0 to any power is always zero - e.g. 03 = 0 x 0 x 0 = 0. So there is no way to represent numbers in base 1 or 0. Its also completely feasible to work with bases greater than 10. Computer science generally works in base 16 where we use the symbols A,B,C,D,E,F to represent the numbers that follow 9. 26 ( base 10 ) is represented as 1A in base 16.
